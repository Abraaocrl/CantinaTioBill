# CantinaTioBill
Aplicação desenvolvida para seleção de vaga de Desenvolvedor do Grupo SD

Para a execução estável da aplicação, clone o repositório. 

Abra o arquivo ApplicationDbContext.cs dentro da pasta de Models da solução.

Altere o "Data Source" da string de conexão para o banco o qual será utilizado.

Execute o comando "update-database" no console de Package Mananger para o banco ser criado pelo Entity Framework.

A aplicação usou SQL Server, as instruções acimas funcionam apenas para máquinas com esse SGBD.
